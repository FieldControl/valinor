# Challenger Field Valinor

## Descrição

Trata-se de uma aplicação em angular e CSS que consome a API do Github para buscar repositórios. 

## Ferramentas Utilizadas

- [Angular](https://angular.io/): É um framework JavaScript criado pelo Google em 2009, para a construção de SPAs. O angular ficou bastante popular e hoje em dia é um dos frameworks mais utilizados no mercado. Na minha aplicação optei pelo angular, apesar de ser muito mais familiarizada em React e NextJS, porque é uma ferramenta que tenho interesse de aprender e um diferencial para o projeto.
- [GitHub API](https://developer.github.com/v3/): Realizei a integração da API HTTP do github para buscar os dados dos repositórios de busca. Mesmo sendo muito nerd e gostar muito da API do Star Wars e dos Pokemons, eu já havia trabalhado com elas em outros projetinhos pessoais. Nunca tinha tido contato com a API do Github, então optei por seguir a ideia do desafio. 
- [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS): Para estilização, apesar de um pouquinho fora dos padrões de mercado hoje, utilizei CSS simples, tentei instalar o Tailwind para utilizar com o Angular mas tive alguns problemas na configuração e estava preocupada com o tempo. Acredito que SASS seria uma alternativa interessante por conta da hereditariedade de classes, o que economizaria código, mas por fim decidi ir pelo CSS.

## Instalação e Execução

1. Clone o repositório: `git clone https://github.com/seu-usuario/seu-repositorio.git`
2. Instale as dependências: `npm install`
3. Execute a aplicação: `ng serve`
4. Acesse em seu navegador: `http://localhost:4200/`

## Recursos e Funcionalidades

A aplicação consiste em apenas uma página (SPA);
Nela há um campo de busca em que o usuário pode digitar o nome de um repositório do github;
A pesquisa funciona depois do usuário digitar no mínimo 3 letras;
Quando feita a requisição, os resultados da pesquisa serão exibidos em cards contendo o link para acessar o repositório, uma descrição breve, o nome do usuário que é dono do repositório, sua imagem de perfil e a quantidade de estrelas.
Os resultados são paginados com 15 itens por página.
