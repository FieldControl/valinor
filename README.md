 ## Projeto Kanban – Angular + NestJS ##

 
Desenvolvido por Maria Julia Paes Ganzella 

## Descrição do projeto: 
Esta aplicação é um sistema de gerenciamento de tarefas (Kanban) desenvolvido como parte de um teste técnico. O objetivo principal é demonstrar habilidades em desenvolvimento Full Stack, com foco em organização de código, consumo de APIs REST e persistência de dados. 

## Principais Funcionalidades: 
- Gerenciamento dinâmico de colunas. 
- Criação e edição de cards (tarefas). 
- Associação de cards a colunas específicas. 
- Persistência de dados com SQLite. 
- Validação de dados de entrada tanto no front quanto no back. 

 
## Tecnologias utilizadas: 
Back-end: 
- Node.js 
- NestJS 
- TypeORM 
- SQLite  
- class-validator 
- Jest (testes) 

Front-end: 
- Angular 
- TypeScript 
- HttpClient 
- HTML 
- CSS 


## Como instalar as dependências: 
Back-end (NestJS) 
- cd backend 
- npm install 
 
Front-end (Angular) 
- cd frontend 
- npm install 

## Como rodar o back-end: 
- cd backend 
- npm run start:dev 
 
## O back-end ficará disponível em: 
http://localhost:3000 

 
## Como rodar o front-end: 
- cd frontend 
- ng serve 

## A aplicação ficará disponível em: 
http://localhost:4200 

 
## Como rodar os testes: 
- cd backend 
- npm run test 

 
## Considerações finais  
O projeto foi estruturado de forma modular, seguindo princípios de Clean Code e separação de responsabilidades. Essa estrutura permite que a aplicação seja facilmente escalável e de simples manutenção
