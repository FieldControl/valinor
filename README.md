## Instalação e configuração.

Faça um clone desse repositório. Tenha instalado `npm` e `node` na sua máquina.

Instalação:

`npm install`

Para iniciar o servidor:

`npm start`

Para rodar os testes:

`npm run test`

## Tecnologias utilizadas
