<h2>🔌 Getting started</h2>
Clone the project:

```bash
$ git clone https://github.com/Goytai/GitCat.git
```

Acess the folder:

```bash
$ cd GitCat
```

Follow the steps below:
```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn start
```
