# Heroes-API
Nodejs REST API exemple using Hapi and Mongoose

# Start
Clone this repo then
```
cd Heroes-API
```
Install depencies;
```
npm i
```
Create a .env file following this exemple;
```
APP_HOST=localhost # Application 
APP_PORT=5000 # Application 

DB_HOST=localhost # Mongo
DB_PORT=27017 # Mongo 
```
Start server;
```
npm start
```
