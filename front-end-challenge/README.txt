# Lista de Repositórios do GitHub

Este é um projeto de exemplo que exibe uma lista paginada de repositórios do GitHub utilizando a API do GitHub. A lista é exibida com recursos de paginação, exibição de detalhes do repositório e estilo personalizado.

## Funcionalidades

- Exibe uma lista paginada de repositórios do GitHub.
- Mostra o nome do repositório, URL, descrição e imagem do proprietário.
- Permite navegar entre as páginas utilizando os botões Anterior e Próximo.
- Permite selecionar uma página específica através da paginação numérica.

## Tecnologias Utilizadas

- Angular
- HTML
- CSS
- API do GitHub

## Pré-requisitos

- Node.js (https://nodejs.org)
- Angular CLI (instalado globalmente) - `npm install -g @angular/cli`

## Como Executar o Projeto

1. Clone o repositório para o seu ambiente local:

git clone <URL_DO_REPOSITÓRIO>

2. Acesse o diretório do projeto:

cd github-listagem

3.Instale as dependências do projeto:
npm install

4.Inicie o servidor de desenvolvimento:

ng serve

5.Abra o navegador e acesse a seguinte URL:
http://localhost:4200
